# Introduction

## First Steps if You Think There’s an Incident

1. Check **ADP Support – General** for any recent announcements.
2. Contact SME or First Contact for the application or system you are concerned about.
   - If unsure who this is, check the **ADP System Ownership and Routing** table.
   - You can also check the **Team Organisation Charts**.
3. Do not be afraid to **escalate** if you are unsure or cannot get an answer.

Contact **Tech Services** via the channel `ADP Support | General`.

## Incident Triage Flowcharts

- For Networking issues → **Network Triage**
- For Dremio issues → **Dremio Triage**
- For EKS issues → **EKS Triage**

A simplified network topology diagram is available under **Platform → Network Topology**.
